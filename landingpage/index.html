Open landscape

Novel 4-layer framework

The metaverse landscape is structured along a 4-layer framework. It groups projects by type and technologies based on experiences, user capabilities and infrastructure. Each technology layer is split up into various components that are required to build Metaverse applications (mApps).

The framework will evolve over time together with the landscape.

How to contribute to metaverse landscape

The metaverse landscape is a community-driven, open-source project. You can contribute in two ways:

  • Contribute by adding content
  • Contribute by evolving the framework

Make a contribution

Follow the README in our GitHub repository for detailed instructions and create a pull request with your contributions. After review, your contributions will be published.
